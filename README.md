## 설치방법
(option 1) 한방에 설치하기
```
sudo ./set0.sh
```
(option 2) 권한 유무에 따라 나눠 설치하기
```
sudo ./need_sudo.sh
./non_sudo.sh
```
(option 3) 한단계씩 직접 설치하기
```
sudo ./install_pkg.sh
sudo ./change_shell.sh
sudo ./install_npm.sh

./git_clone.sh
./create_symlink.sh
./install_vim.sh
./install_vimPlugin.sh
./edit_dconf.sh
./python_virEnv.sh
```
